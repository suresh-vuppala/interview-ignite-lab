## Advanced Tree Problems — DP on Trees & Hard Patterns

This module covers the hardest tree problems that combine trees with dynamic programming, greedy strategies, and advanced techniques. These are **Google L5+ and Meta E5+** level questions.

<br>

## What Makes These "Advanced"?

- **DP on Trees** — State transitions happen during tree traversal (Binary Tree Cameras, Maximum Sum BST)
- **Re-rooting** — Compute answer for every node as root in O(n) using re-rooting DP
- **Binary Lifting** — Answer kth ancestor queries in O(log n) using sparse table on trees
- **Distance problems** — Nodes at distance K, sum of distances in tree

<br>

## Prerequisites

You should be comfortable with: Tree traversals, recursion, LCA, basic DP, and DFS before attempting these problems.
