# LIS Binary Search

## Problem Statement
Solve LIS Binary Search using dynamic programming.

## Approach

### DP Formula
```
Use binary search on tail array
```

### Steps
1. Define DP state
2. Initialize base cases
3. Fill DP table using recurrence
4. Return final result

## Complexity
- Time: O(n) or O(n²)
- Space: O(n)
