# Russian Doll Envelopes

## Problem Statement
Solve Russian Doll Envelopes using dynamic programming.

## Approach

### DP Formula
```
Sort by width, LIS on height
```

### Steps
1. Define DP state
2. Initialize base cases
3. Fill DP table using recurrence
4. Return final result

## Complexity
- Time: O(n) or O(n²)
- Space: O(n)
