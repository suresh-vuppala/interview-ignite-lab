## Problem Statement
Solve Cherry Pickup using dynamic programming.

## Approach

### DP Formula
```
dp[i1][j1][i2] = max paths
```

### Steps
1. Define DP state
2. Initialize base cases
3. Fill DP table using recurrence
4. Return final result

## Complexity
- Time: O(n) or O(n²)
- Space: O(n)
