# Target Sum

## Problem Statement
Solve Target Sum using dynamic programming.

## Approach

### DP Formula
```
dp[sum] = count ways to reach sum
```

### Steps
1. Define DP state
2. Initialize base cases
3. Fill DP table using recurrence
4. Return final result

## Complexity
- Time: O(n) or O(n²)
- Space: O(n)
