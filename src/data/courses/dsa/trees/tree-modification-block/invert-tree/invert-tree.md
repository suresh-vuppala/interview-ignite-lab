# Invert Binary Tree

## Overview
Swap left and right subtrees for all nodes (mirror the tree).

## Problem Statement
Invert a binary tree so left becomes right and right becomes left.

## Approach
- DFS: swap children at each node
- Base case: null nodes
- Simple recursive solution

## Complexity Analysis
- **Time**: O(n)
- **Space**: O(h)
