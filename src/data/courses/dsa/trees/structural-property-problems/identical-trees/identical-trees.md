## Overview
Check if two binary trees are identical (same structure and values).

## Problem Statement
Determine if tree1 and tree2 are identical.

## Approach
- Base cases: both null (true), one null (false)
- Check values match, then recursively check subtrees
- Simple and elegant

## Complexity Analysis
- **Time**: O(min(n1, n2))
- **Space**: O(h)
