## Standard Problems on Trees

Key problem categories:
1. **Traversals:** Inorder, preorder, postorder, level-order, zigzag, boundary
2. **Properties:** Height, diameter, balance check, width
3. **Path Problems:** Path sum, max path sum, LCA, distance
4. **Modifications:** Invert, flatten, serialize/deserialize
5. **Construction:** From traversals, from sorted array
