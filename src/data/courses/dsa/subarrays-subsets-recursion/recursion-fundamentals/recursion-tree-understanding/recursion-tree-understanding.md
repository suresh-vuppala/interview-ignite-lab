Visualize recursion as a tree structure.

## Key Concepts
- Each node is a function call
- Branches are recursive calls
- Leaves are base cases
- Height = recursion depth

## Example: Fibonacci
- fib(4) splits into fib(3) and fib(2)
- Tree shows overlapping subproblems

## Analysis
- Count nodes for time complexity
- Height for space complexity
