Bit masking uses binary operations to manipulate individual bits.

## Operations
- AND (&): Both bits 1
- OR (|): At least one bit 1
- XOR (^): Bits different
- NOT (~): Flip bits
- Left shift (<<): Multiply by 2
- Right shift (>>): Divide by 2

## Applications
- Set/check/toggle bits
- Generate subsets
- Optimize space
