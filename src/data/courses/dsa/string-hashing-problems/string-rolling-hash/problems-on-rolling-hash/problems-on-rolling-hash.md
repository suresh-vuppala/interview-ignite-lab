# Problems on Rolling Hash

Common problems using rolling hash technique.

## Problem Types
1. Find duplicate substrings
2. Longest repeated substring
3. Check if strings are rotations

## Approach
- Compute hash for all substrings
- Use hash map to detect duplicates
- Handle collisions carefully

## Complexity
- Time: O(n) or O(n log n)
- Space: O(n)
