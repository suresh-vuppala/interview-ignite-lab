## Problem
Compare two strings with '#' representing backspace. Return if they're equal.

## Approach
Process from end using two pointers, skip characters based on backspace count.

## Algorithm
1. Start from end of both strings
2. Count backspaces and skip characters accordingly
3. Compare valid characters
4. Return true if both processed completely

## Time Complexity
O(n + m)

## Space Complexity
O(1)
