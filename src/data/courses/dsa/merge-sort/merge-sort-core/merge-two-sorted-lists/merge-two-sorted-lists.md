## Problem Statement
Merge two sorted linked lists into one sorted linked list.

## Approach
Use two pointers to traverse both lists and merge nodes in sorted order.

**Time Complexity**: O(n + m)  
**Space Complexity**: O(1)

## Algorithm
1. Create dummy node as result head
2. Compare nodes from both lists
3. Link smaller node to result
4. Attach remaining nodes
