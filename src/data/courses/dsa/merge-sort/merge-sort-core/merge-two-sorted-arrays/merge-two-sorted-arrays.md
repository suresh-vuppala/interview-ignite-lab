## Problem Statement
Given two sorted arrays, merge them into a single sorted array.

## Approach
Use two pointers to compare elements from both arrays and merge in sorted order.

**Time Complexity**: O(n + m)  
**Space Complexity**: O(n + m)

## Algorithm
1. Initialize two pointers at start of both arrays
2. Compare elements and add smaller to result
3. Add remaining elements from non-empty array
