# Merge Sort vs Heap Sort

## Comparison

| Aspect | Merge Sort | Heap Sort |
|--------|-----------|-----------|
| **Time (All)** | O(n log n) | O(n log n) |
| **Space** | O(n) | O(1) |
| **Stability** | Stable | Unstable |
| **In-place** | No | Yes |
| **Cache** | Better | Worse |

## When to Use
- **Merge Sort**: Need stability, external sorting
- **Heap Sort**: Memory constrained, guaranteed O(n log n)
