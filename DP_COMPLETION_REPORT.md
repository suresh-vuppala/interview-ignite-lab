## Code Format Standards

All implementations follow these standards:
1. **Section Headers**: Clear `// ==================== SOLUTION X ====================` markers
2. **Complexity Comments**: Time and Space complexity on each solution
3. **Minimal Code**: Only essential logic, no verbose explanations
4. **4 Solutions Each**:
   - Solution 1: Recursive (Brute Force) - O(2^n) or O(3^n)
   - Solution 2: Memoization (Top-Down DP) - O(n)
   - Solution 3: Tabulation (Bottom-Up DP) - O(n)
   - Solution 4: Space Optimized - O(1)
